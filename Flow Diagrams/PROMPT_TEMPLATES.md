# Prompt Templates for Diagram Requests

Copy and customize these templates for your diagram requests.

## Flowchart Templates

### Process Flow
```
Create a flowchart for [process/workflow name] from [PRD name]
```

### User Flow
```
Create a flowchart for the [user action/feature] user flow from [PRD name]
```

### System Flow
```
Create a flowchart for the [system process] flow from [PRD name]
```

## Multi-Diagram Requests

### Complete Visualization Set
```
Create the following diagrams from [PRD name]:
1. A flowchart for [process]
2. A decision tree for [decision point]
3. A timeline for [timeline]
```

### Different Perspectives
```
Create a flowchart and a decision tree for [concept] from [PRD name]
```

## With Focus/Constraints

### Focused Request
```
Create a flowchart for [process] from [PRD name]
Focus on: [specific aspect]
```

### Detailed Request
```
Create a flowchart for [process] from [PRD name]
Include: [additional details]
Show: [specific elements]
```
