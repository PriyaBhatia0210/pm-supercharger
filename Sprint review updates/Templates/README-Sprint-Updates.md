# Sprint Update Templates - Quick Start Guide

This folder contains templates and guides for generating Product Review Updates and Growth Sprint Presentation slides from Sprint Planning Meeting notes and other relevant documents.

## Files Overview

### Templates
- **Sprint-Update-Template-Product-Review.md** - Template for generating Product Review Updates
- **Sprint-Update-Template-Growth-Presentation.md** - Template for generating Growth Sprint Presentation slides (2-3 slides per team)

### Guides
- **Sprint-Update-Extraction-Guide.md** - Comprehensive guide on extracting information from various sources

### Examples
- **Example-Sprint-148-Product-Review-Update.md** - Example Product Review Update generated from Sprint 148 planning notes
- **Example-Sprint-148-Growth-Presentation.md** - Example Growth Sprint Presentation slides for Sprint 148

## Quick Usage

### To Generate Product Review Updates:

1. Gather your sources:
   - Sprint Planning Meeting notes
   - Related PRDs (if applicable)
   - Metrics dashboards/reports (if available)
   - Other relevant meeting notes

2. Use the extraction guide to identify key information:
   - Sprint number and dates
   - Sprint goal
   - Completed work
   - Planned work
   - Metrics and results
   - Action items

3. Fill in the Product Review template with extracted information

4. Review the example file for formatting reference

### To Generate Growth Sprint Presentation Slides:

1. Gather your sources (same as above)

2. Extract information for:
   - **Slide 1**: What was done this sprint (completed work, deliverables)
   - **Slide 2**: What's coming up next sprint (sprint goal, planned work)
   - **Slide 3**: Demo (optional - only if there's something to showcase)

3. Use the Growth Presentation template to structure your slides

4. Review the example file for formatting reference
